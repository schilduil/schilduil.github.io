Schilduil Proc Inbreeding

Beschrijving

Schilduil Proc Inbreeding is een rekenmachine voor inteeltcoëfficienten en additief genetische verwantschap (verwantschapsmatrix).

De elementen van de verwantschpasmatrix bevatten de additief genetische verwantschap tussen twee individuen. Dit is een getal tussen 0 en 2.

De elementen op de diagonaal zijn de additief genetische verwantschappen van de individuen met zichzelf. Dit is gelijk aan 1 + de inteeltcoëfficient.

Voor het gebruik van dit programma kan best eerst het artikel over Inteelt Berekenen lezen.

Start Schilduil Proc Inbreeding

Features

  • Afstammingsbestanden kunnen ingelezen worden vanuit het bestandsysteem of van het internet (vanaf versie 1.0)
  • De verwantschapsmatrix kan geexporteerd worden naar een CSV-bestand (vanaf versie 1.0)
