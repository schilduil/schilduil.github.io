Schilduil Proc Inbreeding

Description

The Schilduil Proc Inbreeding is a calculator of inbreedingcoefficients and additive genetic relationships based on pedigree (relationship matrix).

The elements of the relationship matrix represent the additive genetic relationship between two individuals. This is a number between 0 and 2.

The diagonal elements of the relationship matrix represent the additive genetic relationships of an individual with itself. This equals 1 + the inbreeding coefficient of that individual

I suggest you first read the article about Calculation Inbreeding before using the software.

Start Schilduil Proc Inbreeding

Features

  • Can read pedigree files from both the filesystem as from the internet. (since version 1.0)
  • Can export the relationship matrix to a CSV file (since version 1.0)
